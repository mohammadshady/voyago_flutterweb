# Flutter Web Frontend

This is the frontend built with Flutter Web.

## Requirements
- Flutter 3.x
- Chrome browser

## Setup Instructions

1. Clone the repository
   git clone https://github.com/username/flutter-web-repo.git
   cd flutter-web-repo

2. Install packages
   flutter pub get

3. API Configuration


Default:
http://127.0.0.1:8000/api

4. Run the project
   flutter run -d chrome
